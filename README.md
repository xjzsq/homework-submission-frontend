# 作业提交 - 后台前端

这是 [作业提交](http://u.d1.fan) 的后台前端部分，基于 [Ant Design Pro](https://pro.ant.design) 开发。

## 环境

安装 `node_modules`:

```bash
npm install
```

或

```bash
yarn
```

## 可行的脚本

在 `package.json` 中提供的脚本如下：

### 快速开始

```bash
yarn start
```

### 打包工程

```bash
yarn build
```

### 检查代码风格

```bash
yarn lint
```

自动修复代码风格

```bash
yarn lint:fix
```

### 测试代码

```bash
yarn test
```
